# shopping catalog

This is a simple application developed with React Native.


#### 적용 모듈
  - react-native-firebase
  - react-native-paper
  - react-native-router-flux
  - react-native-simple-toast
  - react-native-app-intro-slider
  - react-native-chart-kit
